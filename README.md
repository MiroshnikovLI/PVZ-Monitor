# 📦 PVZ Monitor

**PVZ Auditor Bot** — это автоматизированный сервис для мониторинга прохождения аудитов на пунктах выдачи заказов (ПВЗ). Бот проверяет статус аудитов по заданному списку ПВЗ и отправляет уведомления в чат ВКонтакте при обнаружении изменений или нарушений.

---

## ✨ Возможности

- ✅ Мониторинг статусов аудитов на указанных ПВЗ
- 🔔 Автоматические уведомления в чат ВКонтакте
- ⏱️ Настраиваемая частота проверок
- 🖥️ Веб-админ-панель (HTML + CSS + ванильный JS)
- 🔄 Ручной запуск проверки по требованию
- 📝 История последних проверок и уведомлений
- 🗄️ Хранение настроек в SQLite

---

## 🧱 Технологии

| Компонент | Технология |
|-----------|------------|
| **Бэкенд** | Node.js, Express (чистый JS) |
| **Фронтенд (админка)** | HTML, CSS, ванильный JavaScript |
| **База данных** | SQLite |
| **Парсинг** | Axios, Cheerio |
| **Уведомления** | VK API (vk-io) |
| **Расписание** | node-cron |
| **Сервер** | Ubuntu (VPS) |

> ⚠️ В проекте не используется TypeScript, React, Vue или другие фреймворки — только чистый JavaScript.

## 🚀 Быстрый старт

### Требования

- Node.js 18+
- npm
- Доступ к VPS (или локальная машина для разработки)

### Установка

```bash
# Клонируем репозиторий
git clone https://github.com/yourusername/pvz-auditor-bot.git
cd pvz-auditor-bot

# Устанавливаем зависимости
npm install

# Создаем файл с переменными окружения
cp .env.example .env
# Отредактируйте .env — укажите токен VK, ID чата и другие настройки

pvz-auditor-bot/
├── backend/
│   ├── index.js          # Точка входа API (Express)
│   ├── parser.js         # Логика парсинга сайта
│   ├── notifier.js       # Отправка уведомлений в VK
│   ├── scheduler.js      # Настройка расписания (node-cron)
│   ├── db.js             # Работа с SQLite
│   └── routes.js         # API-маршруты
├── public/
│   ├── index.html        # Админ-панель
│   ├── style.css         # Стили
│   └── app.js            # Ванильный JS для фронтенда
├── .env.example          # Пример переменных окружения
├── package.json
└── README.md

⚙️ Переменные окружения (.env)
Переменная	Описание
VK_ACCESS_TOKEN	Токен доступа VK API
VK_CHAT_ID	ID чата ВКонтакте для уведомлений
CHECK_INTERVAL_MINUTES	Интервал проверок (в минутах)
PORT	Порт для веб-сервера (по умолчанию 3000)
DATABASE_URL	Путь к файлу SQLite
